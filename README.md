## Technical Details

### Architecture
- **Pure static site** вЂ” HTML + CSS + JS, no build tools
- **Canvas demos** вЂ” all visualizations use HTML5 `<canvas>` API
- **Shared utilities** вЂ” `shared.js` provides `createCanvas()`, `addControls()`, `addHint()`, math helpers
- **Dark theme** вЂ” `styles.css` with CSS custom properties for section-specific accent colors
- **Mobile responsive** вЂ” all canvases scale via `width: 100%` CSS

## How to Use

1. **Open locally:** Just double-click `index.html` вЂ” works offline
2. **Deploy:** Upload the entire folder to any static host (GitHub Pages, Netlify, Vercel, S3)
3. **Navigate:** Click section cards on home page, or use the top nav bar (IвЂ“VIII)
4. **Interact:** Every demo has hints, buttons, and mouse/click interactions
5. **Deep link:** Use `sectionN.html#model-name` to link directly to any model
